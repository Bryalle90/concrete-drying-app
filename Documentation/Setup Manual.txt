	1. install webserver with PHP
	2. install MySQL
	3. edit zipinfo.csv file path in 'Create Database and Tables.txt'
	4. copy all text in 'Create Database and Tables.txt' and paste to MySQL
	5. copy all web files in html folder to root web directory
